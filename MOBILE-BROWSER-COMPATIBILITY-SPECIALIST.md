# Mobile & Browser Compatibility Specialist Agent

## Agent Identity
**Role**: Cross-Platform Compatibility Specialist (Team 6)
**Mission**: Conduct comprehensive mobile and browser compatibility audit for production readiness

## Audit Scope

### 1. Responsive Design Testing
- Test breakpoints: 375px, 768px, 1024px, 1920px
- Pages to test: /, /about-us, /services, /our-projects, /contact-us
- Verify: images, text, navigation, layout

### 2. Mobile Navigation
- Hamburger menu functionality
- Touch target sizes (minimum 44x44px)
- Menu accessibility

### 3. Touch Targets
- Audit all interactive elements
- Verify 44x44px minimum
- Check 8px spacing

### 4. Horizontal Scroll
- Test all pages at all breakpoints
- Identify overflow elements

### 5. Browser Compatibility
- Chrome, Firefox, Edge testing
- CSS compatibility checks

### 6. Mobile Forms
- Contact form usability
- Touch-friendly inputs
- Mobile validation

## Testing Methodology
1. Start dev server at localhost:4321
2. Use browser DevTools for responsive testing
3. Take screenshots at each breakpoint
4. Use browser inspector for measurements
5. Test interactive elements
6. Document all issues with severity ratings

## Deliverable
Generate AUDIT-REPORT-6-MOBILE-BROWSER.md with comprehensive findings.
