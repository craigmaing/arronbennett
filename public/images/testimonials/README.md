## Image Specifications
- **Dimensions**: 150x150px minimum (will be displayed at ~80x80px)
- **Format**: JPG or WebP
- **Crop**: Circular/square portrait
- **Quality**: Medium-high (70-80%)
- **File Size**: Target <50KB each

## Fallback Behavior

- First initial in a colored circle
- Navy background with white text
- Automatically generated from name
